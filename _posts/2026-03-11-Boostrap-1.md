# I. Linear Interpolation of Discount Curve
## 0. Forward rate and zero-rate formula <a name="section-0"></a>

The definition of discount factor is as below:

$$
P(T)=\exp\left(-\int_0^T f(u)\,du\right)
$$

The continuous-compounded zero-rate has the following formula:

$$
z(0,T) = -\frac{1}{T}\ln P(0,T)
$$


The instantaneous forward rate has the following formula:

$$
f(0,T) = -\frac{d}{dT} \ln P(0,T)
$$

or we can write it as 

$$
f(0,T) = -\frac{P'(0,T)}{P(0,T)}
$$

or we can write it as (from Brigo & Mercurio)

$$
f(0,T) = -\frac{\partial}{\partial T} \ln P(0,T)
$$

Now we will compare different interpolation schemes between two maturities 

$$
T_1 \lt T \lt T_2
$$

## 1. Log-linear interpolation in discount factor (log-linear DF interpolation) <a name="section-1"></a>

We assume that

$$\ln P(0,T)$$ is linear between $$T_1$$ and $$T_2$$

then 

$$
\ln P(0,T) = a + bT
$$

then 

$$
P(0,T) = e^{a+bT}
$$

from instantaneous forward rate formula, we have

$$
\boxed{
f(0,T) = -\frac{d}{dT} \ln P(0,T) = -b
}
$$

We can get:
- Piecewise constant instantaneous forward
- Smooth, arbitrage-consistent shape
- No oscillations

Formal proof

Assume we have

$$
\ln P(0,T)= (1-w)\ln P(0,T_1)+w\ln P(0,T_2),
\quad w=\frac{T-T_1}{T_2-T_1}.
$$

We differentiate w.r.t $$T$$ with $$w'(T)=\displaystyle\frac{1}{T_2-T_1}$$

$$
\frac{\partial}{\partial T}\ln P(0,T)
= w'(T)\big(\ln P(0,T_2)-\ln P(0,T_1)\big)
= \frac{\ln P(0,T_2)-\ln P(0,T_1)}{T_2-T_1},
$$

which is a constant where no $$T$$ is left. We can rewrite final result as 

$$
\boxed{
f(0,T)= -\frac{\partial}{\partial T}\ln P(0,T)
= -\frac{\ln P(0,T_2)-\ln P(0,T_1)}{T_2-T_1}
}
$$

is constant for all $$T\in[T_1,T_2]$$





## 2. Linear interpolation in discount factor

Now assume that  we have

$$
P(0,T) = (1-w)P_1 + wP_2
$$

where 

$$
w = \frac{T-T_1}{T_2-T_1}
$$

Rearranging a little bit to group $$T$$ together, we get

$$
P(0,T) = P_1 + \frac{P_2 - P_1}{T_2 - T_1} (T-T_1) = 
P_1 - T_1 \frac{P_2 - P_1}{T_2 - T_1} + \frac{P_2 - P_1}{T_2 - T_1}T =
\frac{P_1 T_2 - P_2 T_1}{T_2 - T_1} + \frac{P_2 - P_1}{T_2 - T_1}T
$$

so technically, it's just in the linear form

$$
P(0,T) = \alpha + \beta T
$$

where 

$$
\beta = \frac{P_2 - P_1}{T_2 - T_1}
$$

and 

$$
\alpha = \frac{P_1 T_2 - P_2 T_1}{T_2 - T_1}
$$

From this, we can calculate the forward rate where

$$
f(0,T) = -\frac{P'(0,T)}{P(0,T)}
$$

Here we can clearly see that 

$$
P'(0,T) = \beta = \frac{P_2 - P_1}{T_2 - T_1}
$$

when we get

$$
\boxed{
f(0,T) = -\frac{\beta}{\alpha + \beta T}
}
$$

so the forward rate is in this form:

$$
f(T) = \frac{\text{constant}}{\text{linear function}}
$$


We can get
- A hyperbolic function in $$T$$
- It changes continuously across different time stamp
- The forward curve bends nonlinearly between pillars 


Formal proof

## 3. Linear interpolation in zero-rates

From the formula of zero-rates we have

$$
z(0,T)= -\frac{1}{T}\ln P(0,T)
\quad\Longleftrightarrow\quad
\ln P(0,T)= -T\,z(0,T).
$$

We assume a linear relationship as 

$$
z(0,T)= (1-w)z(0,T_1)+w z(0,T_2),
\quad w=\frac{T-T_1}{T_2-T_1}.
$$


As $$z(0,T)$$ is an affine function in $$T$$, we can write it as:

$$
z(0,T)=a+bT,
$$

Plugging back into $$\ln$$ formula above, we have a quadratic form as

$$
\ln P(0,T)= -T(a+bT)=-(aT+bT^2).
$$

Now differentiate with resepect to $$T$$ to get the instantaneous forward rate is

$$
f(0,T)= -\frac{\partial}{\partial T}\ln P(0,T)= a+2bT,
$$

which is linear form in $$T$$ with $$b \neq 0$$

## 4. Linear interpolation in the logarithm of zero rate

Using continous compounding where

$$
P(T) = e^{-T z(T)},
$$

we have

$$
\ln P(T)=-T z(T)
$$

Using product-rule w.r.t $$T$$, we got

$$
f(T)= -\frac{d}{dT}\ln P(T)=\frac{d}{dT}[T z(T)] = z(T) + Tz'(T).
$$

Assuming we have interpolation on $$\ln z(T)$$ we got

$$
y(T):=\ln z(T) = y_1+\frac{y_2-y_1}{T_2-T_1}(T-T_1),
$$

where 

$$
y_1=\ln z_1,\qquad y_2 = \ln z_2
$$

and 

$$
z_1=z(T_1),\qquad z_2=z(T_2).
$$

We can define coefficient $$b$$ as 

$$
b:=\frac{\ln z_2-\ln z_1}{T_2-T_1}.
$$

Then we got

$$
\ln z(T)=\ln z_1+b(T-T_1)
$$

We can exponentiate to get cleaner form where zero rate is exponential in maturity on that interval:

$$
z(T) = z_1 e^{b(T-T_1)}
$$

We can differentiate the exponential form

$$
z'(T) = bz(T)
$$

The instantaneous forward rate can be re-written as

$$
f(T)=z(T)+Tbz
$$

and we got the final form 

$$
\boxed{
f(T)=z_1 e^{\,b(T-T_1)}(1+bT)
}
$$

we can remap $$b$$ as 

$$
\boxed{
f(T)=z_1 e^{\displaystyle\frac{\ln z_2-\ln z_1}{T_2-T_1}(T-T_1)}\left[1+\displaystyle\frac{\ln z_2-\ln z_1}{T_2-T_1}T\right]}
$$

Recall that with continuous zero-rate, we have

$$
z(T)=z_1\left(\frac{z_2}{z_1}\right)^{\frac{T-T_1}{T_2-T_1}}
$$

the formula above can be expressed in a different way as 

$$
\boxed{
f(T)=
z_1\left(\frac{z_2}{z_1}\right)^{\frac{T-T_1}{T_2-T_1}}
\left[
1+
T\frac{\ln(z_2/z_1)}{T_2-T_1}
\right]
}
$$

## Summary

| Method      | Interpolated Quantity | Forward Shape |
| ----------- | --------------------- | ------------- |
| Log DF      | (\ln P)               | Constant      |
| Linear DF   | (P)                   | Hyperbolic    |
| Linear Zero | (z)                   | Linear        |
| Log Zero    | (\ln z)               | Nonlinear     |


# II. Numerical illustration of different interpolation schemes

Assuming the quant guy extracts from his calibration library and gives us 2 data points from discount curve below:

$$
P(0,1)=0.95,\qquad P(0,2)=0.90,
$$

Our task is to calculate the instantaneous forward rate $$f(0,T)$$ using 4 different interpolation schemes we discussed at $$T=1.5$$.

We got:

$$
\ln(0.95)=-0.0512932944
$$

and 

$$
\ln(0.90) = -0.1053605157
$$


From 2 discount factors, we can calculate continuously compounded zero-rates:

$$
z(0,1) = -\ln(0.95) = 0.0512932944 \approx 0.0513
$$

$$
z(0.2) = -\displaystyle\frac{1}{2}\ln(0.90) = 0.0526802578 \approx 0.0527
$$

**1. Log-linear interpolation in discount factor**

The slope is:

$$
f(0,T) = -\frac{d}{dT}\ln P(0,T)=- \frac{\ln(0.90)-\ln(0.95)}{2-1} =  0.1053605157-(-0.0512932944)=-0.0540672213 = 0.0540672213 

$$

So for every $$T\in[1,2]$$, the forward rate is flat at approximate $$5.40672\%$$

**2. Linear interpolation in discount factor**

We got

$$
P(0,T)=0.95 + \frac{0.90 - 0.95}{2 - 1}(T-1) = 1 - 0.05T
$$

set $$T=1.5$$ we got

$$
P(0,1.5)=1-0.05(1.5)=0.925
$$

Apply the instantaneous forward rate formula

$$
f(0,T)=-\frac{P'(0,T)}{P(0,T)}
$$

with 
$$
\beta = P'(0,T)=-0.05
$$

We finally got

$$
f(0,1.5)=\frac{0.05}{0.925}=0.0540540541
$$

**3. Linear interpolation on zero-rate**

We interpolate $$z(0,T)$$ between $$z_1$$ and $$z_2$$

$$
z(0,T)=z_1+\frac{z_2-z_1}{2-1}(T-1)
$$

The slope is 

$$
b=z_2-z_1=0.0526802578-0.0512932944=0.0013869634
$$

hence 

$$
z(0,T)=0.0512932944+0.0013869634(T-1)
$$

To get the instantaneous forward, we use the formula 

$$
f(0,T)=z(0,T)+T\,z'(0,T)
$$

Finally we have

$$
\boxed{f(0,1.5)=0.0540672213=5.406722\%}
$$

**4. Linear interpolation in the logarithm of zero-rate**

First we calculate

$$
\ln z_1=\ln(0.0512932944)\approx -2.9704041590
$$

$$
\ln z_2=\ln(0.0526802578)\approx -2.9437217773
$$

We can calculate the slope using

$$
b=\ln z_2-\ln z_1\approx 0.0266823817
$$

The zero-rate can be derived using

$$
\ln z(0,T)=\ln z_1+b(T-1)
$$

$$
z(0,T)=z_1 e^{b(T-1)}
$$

so we have 

$$
\ln z(0,T)=\ln z_1+b(T-1)
$$

Then 

$$
z(0,1.5)=z_1 e^{0.5b}
= \sqrt{z_1 z_2}
=\sqrt{0.0512932944\cdot 0.0526802578}
\approx 0.0519821503
$$

We have

$$
f(0,T)=z(0,T)+T z'(0,T)=z(0,T)(1+bT)
$$

At $$T=1.5$$ we get

$$
f(0,1.5)=0.0519821503\cdot (1+1.5\times 0.0266823817)
$$

Finally we got

$$
\boxed{f(0,1.5)=0.0540622524=5.406225\%}
$$

**Final result**

| Interpolation scheme       | (P(0,1.5)) or (z(0,1.5)) |    (f(0,1.5)) |
| -------------------------- | -----------------------: | ------------: |
| 1. Log-linear on DF        |  (P(0,1.5)=0.9246621004) | **5.406722%** |
| 2. Linear on DF            |  (P(0,1.5)=0.9250000000) | **5.405405%** |
| 3. Linear on zero rate     |     (z(0,1.5)=5.198678%) | **5.406722%** |
| 4. Linear on log zero rate |     (z(0,1.5)=5.198215%) | **5.406225%** |

# III. Coding interpolation schemes for Discount Curve

## 1. Simple Example
We can quickly write a program to calculate all the nodes with 0.1 incremental time-step for 4 interpolation schemes above and compare the results.

```R
# Nodes data
T1 <- 1.0
T2 <- 2.0
P1 <- 0.95
P2 <- 0.90

# Continous-compounded zero rates at nodes
z1 <- -log(P1) / T1
z2 <- -log(P2) / T2

# Grid from 1.1 to 1.9
T_grid <- seq(1.1, 1.9, by = 0.1)

# 1.Log-linear interpolation on DF
f_log_df <- function(T) {
    slope_lnP <- (log(P2) - log(P1)) / (T2 - T1)
    return(-slope_lnP)
}

# 2. Linear interpolation on DF
f_lin_df <- function(T) {
    slope_P <- (P2 - P1) / (T2 - T1)
    P_T <- P1 + slope_P * (T - T1)
    return(-slope_P / P_T)
}

# 3. Linear interpolation on zero-rate
f_lin_zero <- function(T) {
    b <- (z2 - z1) / (T2 - T1)
    z_T <- z1 + b*(T - T1)
    return(z_T + T * b)
}

# 4. Linear interpolation on log zero rate
f_log_zero <- function(T) {
    b <- (log(z2) - log(z1)) / (T2 - T1)
    z_T <- z1 * exp(b * (T - T1))
    return(z_T * (1 + b * T))
}

# 5. Result
result <- data.frame(
    T = T_grid,
    f_log_linear_DF = sapply(T_grid, f_log_df),
    f_linear_DF = sapply(T_grid, f_lin_df),
    f_linear_zero = sapply(T_grid, f_lin_zero),
    f_log_zero = sapply(T_grid, f_log_zero)
)
```

```r
# Build grid from 1 month to 30 years
T_grid <- seq(1 / 12, 30, by = 1 / 12)

# We have to rewrite above function to adapt to algorithm on full grid
find_interval <- function(T, pillars) {
    if (T < min(pillars) || T > max(pillars)) {
        stop("T outside pillar range")
    }
    if (T == max(pillars)) {
        return(length(pillars) - 1)
    }
    i <- max(which(pillars <= T))
    if (i == length(pillars)) i <- i - 1
    return(i)
}

#  log-linear interpolation on DF
f_log_df <- function(T, pillars, dfs) {
    i <- find_interval(T, pillars)
    T1 <- pillars[i]; T2 <- pillars[i+1]
    P1 <- dfs[i];    P2 <- dfs[i+1]
    
    slope_lnP <- (log(P2) - log(P1)) / (T2 - T1)
    return(-slope_lnP)
}

#  linear interpolation on DF
f_lin_df <- function(T, pillars, dfs) {
    i <- find_interval(T, pillars)
    T1 <- pillars[i]; T2 <- pillars[i+1]
    P1 <- dfs[i];    P2 <- dfs[i+1]
    
    slope_P <- (P2 - P1) / (T2 - T1)
    P_T <- P1 + slope_P * (T - T1)
    
    return(-slope_P / P_T)
}

#  linear interpolation on zero rate
f_lin_zero <- function(T, pillars, zero_rates) {
    i <- find_interval(T, pillars)
    T1 <- pillars[i]; T2 <- pillars[i+1]
    z1 <- zero_rates[i]; z2 <- zero_rates[i+1]
    
    b <- (z2 - z1) / (T2 - T1)
    z_T <- z1 + b * (T - T1)
    
    return(z_T + T * b)
}

# linear interpolation on log zero rate
f_log_zero <- function(T, pillars, zero_rates) {
    i <- find_interval(T, pillars)
    T1 <- pillars[i]; T2 <- pillars[i+1]
    z1 <- zero_rates[i]; z2 <- zero_rates[i+1]
    
    b <- (log(z2) - log(z1)) / (T2 - T1)
    z_T <- z1 * exp(b * (T - T1))
    
    return(z_T * (1 + b * T))
}
```
